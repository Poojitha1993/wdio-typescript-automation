# 🚀 WDIO TypeScript Automation Framework

## 📌 About Project
This project is an automation framework built using WebdriverIO and TypeScript.  
It covers form automation, validations, and negative test scenarios.

---

## 🧰 Tech Stack
- WebdriverIO
- TypeScript
- Node.js
- Page Object Model (POM)

---

## ✨ Features
- ✅ Data-driven testing
- ✅ Form validation automation
- ✅ Negative scenarios handling
- ✅ UI automation
- ✅ Reusable Page Objects
